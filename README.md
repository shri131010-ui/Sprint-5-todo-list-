# sprint5--react-task-management-board

React Task Management Board - Trello Style

A dynamic Trello-style Task Management Board built with **React.js** and **Vite**. This project demonstrates component-based architecture, state management using `useState`, and dynamic task manipulation through React.

## 🔗 Important Links

**Live Website:** https://sprint-5-todo-list.vercel.app/

**Demo Video:** https://drive.google.com/file/d/1YfBydyOP_nv7yHUuCLTcCkta7yIc111w/view?usp=drivesdk


##  Features

* Three-column Task Board (To Do, In Progress, Done)
* Add new tasks using an input field
* Delete tasks from any column
* Move tasks between columns
* State management using React `useState`
* Responsive and clean user interface

##  Tech Stack

* React.js
* Vite
* JavaScript (ES6+)
* CSS3
* React Hooks (`useState`)
* Component-Based Architecture
